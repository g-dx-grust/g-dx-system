## 1. はじめに

G-DX は、グラスト社の営業活動を一元管理するビジネス管理システムです。
以下の2つの事業部に対応しています。

| 事業部コード | 名称 |
|---|---|
| LARK_SUPPORT | Lark導入支援 |
| WATER_SAVING | 節水器具販売（JET事業） |

### 利用できる主な機能

- **顧客管理** — 会社・コンタクト情報の登録・検索・更新
- **営業管理** — 案件のパイプライン管理、ステージ追跡、契約管理
- **コールシステム** — コールキュー管理、架電記録、履歴確認
- **ダッシュボード** — KPI進捗、チーム活動状況、次回アクション確認
- **節水事業（JET）** — 施設管理、JET契約管理（WATER_SAVING事業部のみ）

> **注意:** 表示される機能はご自身のロール（役職権限）と所属事業部によって異なります。

## 2. ログイン・ログアウト

### ログイン

1. ブラウザでシステムのURLにアクセスします。
2. 「**Larkでログイン**」ボタンをクリックします。
3. Larkの認証画面が開くので、会社のLarkアカウントでログインします。
4. 認証が完了すると、自動的にダッシュボードへ遷移します。

> Larkアカウントをお持ちでない場合は、システム管理者にお問い合わせください。

### 事業部の切り替え

複数の事業部に所属している場合、画面上部または設定メニューから事業部を切り替えられます。
切り替え後は、表示されるデータ（案件・契約・コールなど）が選択した事業部のものに絞り込まれます。

### ログアウト

右上のユーザーアイコンをクリックし、「**ログアウト**」を選択します。

## 4. ダッシュボード

### 4.1 個人ダッシュボード

自分のKPI進捗状況と今後のアクション予定を確認できます。

**KPI進捗カード**

| 項目 | 内容 |
|---|---|
| コール数 | 当月の架電件数（実績 / 目標） |
| 訪問数 | 当月の訪問件数（実績 / 目標） |
| アポ数 | 当月のアポイント獲得数（実績 / 目標） |
| 商談数 | 当月の商談実施数（実績 / 目標） |
| 契約数 | 当月の契約件数（実績 / 目標） |
| 売上 | 当月の売上金額（実績 / 目標） |

各カードには達成率（%）が表示されます。

**次回アクション一覧**

- **期限超過** — 対応が遅れているアクション（赤表示）
- **本日** — 今日が期限のアクション
- **今週** — 今週中に対応が必要なアクション

各アクションには案件名・会社名・ステージ・金額・次回アクション内容が表示されます。
案件名をクリックすると案件詳細ページへ移動できます。

### 4.2 案件ダッシュボード

チーム全体の案件状況を俯瞰できます。

**主な表示内容**

- 案件総件数・グループ別集計（アクティブ / 停滞中 / 契約済み）
- ステージ別パイプライン（件数・金額）
- 次回アクション予定リスト（本日・明日・今週）
- 担当者別・会社別の集計表

### 4.3 入金ダッシュボード

契約のステータス別件数・金額サマリーを確認できます。

| ステータス | 内容 |
|---|---|
| 契約済み | 契約が締結された案件 |
| 請求済み | 請求書を発行した案件 |
| 入金済み | 入金が確認された案件 |
| サービス開始 | サービス提供中の案件 |
| サービス終了 | サービスが終了した案件 |

### 4.4 活動ダッシュボード

チームメンバーごとの活動状況（訪問・コール・アポ等）を確認できます。
メンバータブで切り替えながらそれぞれの活動状況を確認します。

### 4.5 KPI目標設定

毎月の個人KPI目標を設定します。

**設定手順**

1. サイドバーから「**KPI目標設定**」を開きます。
2. 対象月を選択します。
3. 各KPI項目（コール・訪問・アポ・商談・契約・売上）に目標値を入力します。
4. 「**保存**」ボタンをクリックします。

## 5. 顧客管理（会社・コンタクト）

### 5.1 会社一覧

サイドバーの「**顧客管理 > 会社**」をクリックします。

**検索・絞り込み**

- 検索ボックスに会社名を入力してリアルタイム検索できます。

**一覧表示項目**

| 列名 | 内容 |
|---|---|
| 会社名 | クリックで詳細ページへ遷移 |
| 業種 | 業種カテゴリ |
| 電話番号 | 直接電話できる場合があります |
| 担当者 | 担当営業担当者名 |
| タグ | 管理用タグ |

### 5.2 会社の新規登録

1. 会社一覧画面右上の「**+ 新規会社**」ボタンをクリックします。
2. 以下の項目を入力します。

| 項目 | 必須 | 説明 |
|---|---|---|
| 会社名 | ✅ | 正式名称を入力 |
| 業種 | — | プルダウンから選択 |
| 電話番号 | — | ハイフンなしで入力 |
| ウェブサイト | — | URLを入力 |
| 担当者 | — | 担当営業を選択 |
| 住所 | — | テキスト入力 |
| タグ | — | 複数設定可能 |

3. 「**保存**」をクリックします。

### 5.3 会社詳細

会社名をクリックすると詳細画面が開きます。

**基本情報カード**

会社の基本情報（業種・電話・ウェブサイト・担当者・住所・タグ）が表示されます。
「**編集**」をクリックすると編集フォームが開きます。

**関連案件カード**

この会社に紐づく直近10件の案件が表示されます。
「**すべて表示**」をクリックすると、この会社で絞り込んだ案件一覧が開きます。

**コンタクト一覧**

この会社の担当者一覧が表示されます。
「**コンタクトを追加**」をクリックすると入力フォームが開きます。

| 項目 | 必須 | 説明 |
|---|---|---|
| 氏名 | ✅ | 担当者名 |
| 部署 | — | 部署名 |
| 役職 | — | 役職名 |
| メールアドレス | — | ビジネスメール |
| 電話番号 | — | 直通番号 |
| 主担当 | — | メインコンタクトの場合チェック |

## 6. 営業管理（案件・契約）

### 6.1 案件一覧

サイドバーの「**営業管理 > 案件**」をクリックします。

**検索・絞り込み**

- キーワード検索（案件名・会社名など）
- ステージでの絞り込み（プルダウン）

**一覧表示項目**

| 列名 | 内容 |
|---|---|
| 案件名 | クリックで詳細ページへ遷移 |
| 会社 | 紐づく会社名 |
| ステージ | 現在の営業ステージ（色付きバッジ） |
| 金額 | 見込み金額（¥表示） |
| 担当者 | 担当営業担当者名 |
| 想定クローズ日 | 契約予定日 |

### 6.2 案件ステージ

案件は以下の7つのステージで管理されます。

| ステージ | 日本語表示 | 意味 |
|---|---|---|
| APO_ACQUIRED | アポ獲得 | アポイントが取れた状態 |
| NEGOTIATING | 商談中・見積提示 | 商談を進めている状態 |
| ALLIANCE | アライアンス | 提携交渉中 |
| PENDING | ペンディング | 一時保留中 |
| APO_CANCELLED | アポキャン | アポイントがキャンセルされた |
| LOST | 失注・不明 | 失注または追跡不明 |
| CONTRACTED | 契約済み | 契約が完了した状態 |

### 6.3 案件の新規作成

1. 案件一覧画面右上の「**+ 新規案件**」ボタンをクリックします。
2. 以下の項目を入力します。

| 項目 | 必須 | 説明 |
|---|---|---|
| 案件名 | ✅ | わかりやすい案件名を入力 |
| 会社 | ✅ | 紐づける会社を選択 |
| 金額 | — | 見込み金額（円） |
| 想定クローズ日 | — | 契約予定日 |
| ソース | — | 案件の発生元 |
| 獲得方法 | — | インバウンド・展示会など |
| 次回アクション日 | — | 次に対応する日付 |
| 次回アクション内容 | — | 次回対応の内容 |
| メモ | — | 補足情報 |

3. 「**保存**」をクリックします。

### 6.4 案件詳細

案件名をクリックすると詳細画面が開きます。

**ステージ変更**

詳細画面右側（またはページ内）のステージ変更パネルから、現在以外のステージのボタンをクリックすることでステージを変更できます。
変更履歴は「**ステージ変更履歴**」タイムラインに自動記録されます。

**案件情報の編集**

「**案件情報を編集**」をクリックしてフォームを開き、各項目を更新して「**保存**」します。

**活動の記録**

案件詳細画面の右カラム（または下部）にある活動ログから、新しい活動を記録できます。

| 項目 | 内容 |
|---|---|
| 活動タイプ | VISIT（訪問）/ ONLINE / CALL / EMAIL / OTHER |
| 活動日 | 実施した日付 |
| サマリー | 活動内容のメモ（任意） |

「**記録**」ボタンをクリックして保存します。

**Lark連携設定**

案件詳細画面の「**Lark連携設定**」アコーディオンからLarkとの連携を設定できます。

| 項目 | 形式 | 説明 |
|---|---|---|
| グループチャットID | oc_xxxxxxxxxx | 案件専用のLarkグループチャットID |
| カレンダーID | — | Larkカレンダーのシンクに使用 |

### 6.5 契約一覧・詳細

サイドバーの「**営業管理 > 契約**」をクリックします。

**検索・絞り込み**

- キーワード検索
- ステータスでの絞り込み

**契約ステータス**

| ステータス | 意味 |
|---|---|
| 契約済み（CONTRACTED） | 契約書締結完了 |
| 請求済み（INVOICED） | 請求書発行済み |
| 入金済み（PAID） | 入金確認済み |
| サービス開始（SERVICE_STARTED） | サービス提供中 |
| サービス終了（SERVICE_ENDED） | サービス終了 |

**契約の新規作成**

1. 「**+ 新規契約**」ボタンをクリックします。
2. 契約情報（タイトル・契約番号・会社・金額・契約日など）を入力します。
3. 「**保存**」をクリックします。

## 7. コールシステム

### 7.1 コールキュー（コールキュー）

架電予定のリストを管理します。

**架電先の追加**

1. 「**+ 追加**」ボタンをクリックします。
2. 以下の項目を入力します。

| 項目 | 必須 | 説明 |
|---|---|---|
| 会社 | ✅ | 架電先の会社を選択（電話番号が自動入力される） |
| 電話番号 | ✅ | 確認・修正可能 |
| 予定日時 | — | 架電予定の日時 |
| メモ | — | 架電前の確認事項など |

3. 「**追加**」をクリックします。

**架電の実施と記録**

1. キューテーブルの「**架電済み**」ボタンをクリックします。
2. 架電先情報が表示されます。Zoom Phone連携がある場合は電話番号リンクから発信できます。
3. 通話後、結果ボタン（7種類）から該当するものを選択します。

**コール結果の種類**

| 結果 | 説明 | ワンクリック完了 |
|---|---|---|
| 不通 | 電話がつながらなかった | ✅ |
| 代理対応 | 受付や代理の方が対応 | — |
| 代理拒否 | 受付に断られた | ✅ |
| KM対応 | キーマン（決裁者）が対応 | — |
| KM拒否 | キーマンに断られた | ✅ |
| アポ獲得 | アポイントが取れた | — |
| 資料送付 | 資料を送ることになった | — |

- **ワンクリック完了** の結果は、ボタンをクリックするだけで記録が完了します。
- その他の結果では、「次回架電予定日時」「メモ」を入力してから「**記録して次へ**」をクリックします。

**過去のコール履歴（サイドバー）**

架電記録画面の右側に、同じ会社への過去のコール履歴が表示されます。
結果・日時・担当者・メモを確認できます。

### 7.2 コール履歴（コール履歴）

過去の架電記録を一覧で確認できます。

**検索・絞り込み**

- 会社名で検索
- コール結果で絞り込み

**サマリーカード**

画面上部に結果種別ごとの件数カードが表示されます。

**履歴テーブル**

| 列名 | 内容 |
|---|---|
| 架電日時 | 日本時間で表示 |
| 会社 | クリックで会社詳細へ遷移 |
| コンタクト | 対応したコンタクト名 |
| 結果 | 色付きバッジで表示 |
| メモ | 架電時のメモ |
| 担当者 | 架電した担当者名 |

**架電記録の手動追加**

画面上部の「**+ 架電記録を追加**」ボタンから、キューを使わずに直接架電記録を追加できます。

## 9. よくある質問

**Q. ログインできません**
A. Larkにログインしていることを確認してください。Larkアカウントが会社のワークスペースに紐づいていない場合は、管理者にお問い合わせください。

**Q. 特定のメニューが表示されません**
A. お持ちのロール（権限）によって表示される機能が異なります。必要な機能へのアクセスが必要な場合は、管理者にロールの変更を依頼してください。

**Q. 節水事業（JET）のメニューが表示されません**
A. WATER_SAVING事業部へのメンバーシップが必要です。管理者に事業部への追加を依頼してください。

**Q. 案件のステージを誤って変更してしまいました**
A. ステージ変更履歴タイムラインで変更記録を確認できます。正しいステージに再度変更してください。

**Q. コール結果を間違えて記録しました**
A. コール履歴から記録を確認し、管理者または担当マネージャーに修正を依頼してください。

**Q. KPI目標が保存できません**
A. 対象月と各項目に正しい数値が入力されているか確認してください。数値は半角数字で入力してください。

**Q. データのCSVインポート/エクスポートはできますか？**
A. インポート機能（会社データなど）はオペレーター以上の権限で利用できます。エクスポート機能はロールによって利用範囲が異なります。管理者にお問い合わせください。
